# AquaCare

Do you care for your aquarium ? This project is for you !

## Uses cases

* Handle Aquarium
* Handle Aquarium Measures (pH, kH, GH, po4, sio2, ...) whit Date records
* SOON - Handle Aquarium Plants
* SOON - Handle Aquarium Live ( fish, invertebrate, ...)
